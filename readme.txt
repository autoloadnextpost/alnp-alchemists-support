### Install

Install and Activate the Child Theme under "Appearance > Themes".

### Auto Load Next Post Settings

Go to "Settings > Auto Load Next Post" to enter the following theme selectors and then press "Save Changes".

Content Container: div#primary

Post Title: h2.post__title

Post Navigation: div.post-related__prev

Comments Container: div#comments


That's it. Now you will see posts auto load.

Enjoy!
